This Week In Servo 26

(2015-03-04) What's up with Servo for the week of 25 Feb - 3 Mar 2015

This week, we merged 50 pull requests

###Notable additions

  • Josh added support for async document loading
  • Patrick implemented overflow-x/overflow-y
  • Patrick and Simon implemented image-rendering and background-size
  • Glenn fixed iframe visibility
  • James added support for beforescriptexecute and afterscriptexecute
  • Ms2ger added profiling to image decoding
  • Glenn added the ability to reap layout data when a node is removed

###New contributors

  • Arbitrary Cat
  • Avi Weinstock
  • Deepak
  • dhneio
  • Isaac Ge
  • James Gilbertson

###Meeting

Minutes

  • We discussed possibilities for good introductory layout issues
  • We’re working on tracking down memory leaks
  • If you want to work with Servo, there’s a Staff Research Engineer position open!
  • Sadly, Mozilla was not accepted into GSoC this year. We discussed alternatives
